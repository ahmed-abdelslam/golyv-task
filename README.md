### Instructions
- **composer install**
- **cp .env.example .env**
- **update DB credits in .env**
- **php artisan key:generate**
- **php artisan migrate**
- **php artisan db:seed**
- **npm install**
- **npm run dev**
- **php artisan serve** In Another Terminal
